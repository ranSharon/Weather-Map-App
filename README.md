# Weather-Map-App
* clone this repository
```bash
    git clone https://github.com/ranSharon/Weather-Map-App.git
```
* open repository
```bash
    cd Weather-Map-App
```
* install with npm 
```bash
    npm i
```
* run the application with 
```bash
    npm start
 ```   
* open your browser at the address http://localhost:8080
